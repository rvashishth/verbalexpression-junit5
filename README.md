# verbalexpression-junit5
Contains example of verbalexpression library with junit5 test cases

###### Pre-requisite/environment
  - Java 1.9
  - Eclipse Photon
  - Junit5 
    - jupiter engine 5.1.0 
    - platform launcher 1.2.0
  - Verbal Expression 1.5
